## Installation
~~~
$ git clone git@github.com:rmrs/react-native-settings.git  
$ cd example
$ npm install
~~~

iOS:
~~~
$ react-native run-ios
~~~

Android
~~~
$ react-native run-android
$ npm start
~~~
